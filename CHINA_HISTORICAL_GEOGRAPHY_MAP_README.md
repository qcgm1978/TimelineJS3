# 中国历史地理分区立体地形图

## 项目概述
这是一个结合历史区域命名的中国地理分区立体地形图可视化页面，旨在直观展示中国地理环境与历史发展的空间关系。

## 使用方法

### 1. 直接在浏览器中打开

**在 macOS 系统中：**
```bash
open china-historical-geography-map.html
```

**在 Windows 系统中：**
```cmd
start china-historical-geography-map.html
```

**在 Linux 系统中：**
```bash
xdg-open china-historical-geography-map.html
```

### 2. 或使用本地服务器（推荐）
```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```
然后在浏览器中访问：`http://localhost:8000/china-historical-geography-map.html`

## 功能介绍

### 1. 交互式地图
- **立体地形图层**：使用 OpenTopoMap 提供的地形数据，展示中国的立体地形
- **多种地图类型切换**：支持地形地图、卫星地图和普通地图三种视图
- **缩放与平移**：使用鼠标滚轮缩放，拖动平移
- **点击查看详情**：点击地图上的区域可查看详细信息

### 2. 地图图层控制
- **自然地理单元**：显示蒙古高原、东北平原、青藏高原、云贵高原等自然地理区域
- **历史文化区域**：显示西域、河西走廊、关中、巴蜀、中原、江南等历史文化区域
- **地理通道**：显示河西走廊等关键地理通道

### 3. 控制按钮
- **自然地理单元**：切换显示/隐藏自然地理区域
- **历史文化区域**：切换显示/隐藏历史文化区域
- **地理通道**：切换显示/隐藏地理通道
- **放大**：放大地图
- **缩小**：缩小地图
- **居中**：将地图重置为初始居中位置
- **切换地图类型**：在地形地图、卫星地图和普通地图之间切换

## 核心内容

### 1. 自然地理单元
- **蒙古高原**：亚洲中部内陆高原，中国重要的草原牧区和生态屏障
- **东北平原**：中国最大的平原，土壤肥沃，重要的粮食生产基地
- **青藏高原**：世界屋脊，平均海拔4000米以上，长江、黄河等大江大河的发源地
- **云贵高原**：地形崎岖，喀斯特地貌发育，生物多样性丰富

### 2. 历史文化区域
- **西域**：今新疆地区，古代丝绸之路的重要组成部分
- **河西走廊**：连接中原与西域的咽喉要道，古代丝绸之路的关键路段
- **关中**：今陕西中部，渭河平原，古代多个王朝的建都之地
- **巴蜀**：今四川盆地，依托都江堰水利工程，历史上的"天府之国"
- **中原**：今河南一带，华夏文明的核心发源地之一
- **江南**：今长江中下游地区，自唐宋后逐渐发展为经济重心

### 3. 地理通道
- **河西走廊**：连接中原与西域的关键通道，古代丝绸之路的咽喉要道

## 自定义修改

### 1. 添加新的地理区域
在 `china-historical-geography-map.html` 文件中，找到对应的数组（`naturalRegions`、`historicalRegions` 或 `passages`），按照以下格式添加新的区域：

```javascript
{
    name: '区域名称',
    coordinates: [[纬度1, 经度1], [纬度2, 经度2], [纬度3, 经度3], ...], // 多边形坐标
    description: '区域描述',
    history: '历史意义' // 可选
}
```

### 2. 修改地图样式
在 CSS 部分可以修改以下样式：
- 地图容器大小：`#interactive-map` 选择器
- 按钮样式：`button` 选择器
- 图例样式：`.legend` 相关选择器
- 弹窗样式：`.info-popup` 相关选择器

### 3. 调整地图初始视图
在 `initMap()` 函数中，修改以下代码：
```javascript
map = L.map('interactive-map').setView([35.8617, 104.1954], 4);
```
其中 `[35.8617, 104.1954]` 是初始中心坐标，`4` 是初始缩放级别。
