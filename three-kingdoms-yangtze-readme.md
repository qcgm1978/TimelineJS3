# 三国时期长江流域地图

## 项目概述
这是一个展示三国时期长江流域江东、江西、江南三大区域的交互式地图，旨在直观展示三国时期长江流域的地理格局和行政区划。

## 使用方法

### 1. 直接在浏览器中打开
```bash
# 在 macOS 系统中：
open three-kingdoms-yangtze-map.html

# 在 Windows 系统中：
start three-kingdoms-yangtze-map.html

# 在 Linux 系统中：
xdg-open three-kingdoms-yangtze-map.html
```

### 2. 或使用本地服务器（推荐）
```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```
然后在浏览器中访问：`http://localhost:8000/three-kingdoms-yangtze-map.html`

## 功能介绍

### 1. 交互式地图
- **区域展示**：清晰标示江东、江西、江南三大历史区域
- **长江水道**：用特殊标记展示长江河道，重点突出三国时期重要的江段，包括荆州地区和吴国核心区域
- **重要城市**：标注三国时期的重要城市，如建业、南昌、苏州等
- **点击查看详情**：点击地图上的区域或城市可查看详细历史信息
- **图例切换**：点击图例可切换显示/隐藏对应的地理要素

### 2. 区域说明
- **江东 (Dongjiang)**：长江下游以东地区，大致包括今江苏南部、浙江北部、安徽东南部，为孙吴政权核心区域
- **江西 (Jiangxi)**：长江以西、赣江流域为中心的区域，大致包括今江西省大部，以豫章郡（南昌）为核心
- **江南 (Jiangnan)**：长江中下游以南的广阔区域，包括今江苏南部、浙江、江西、湖南、安徽南部等

## 核心内容

### 1. 江东区域
- **地理范围**：长江下游以东地区，大致包括今江苏南部、浙江北部、安徽东南部
- **行政区域**：三国时期属孙吴管辖，主要包括江东六郡：吴郡、会稽郡、丹阳郡、豫章郡、庐陵郡、庐江郡
- **历史意义**：孙吴政权的核心根据地，经济发达，人口稠密

### 2. 江西区域
- **地理范围**：长江以西、赣江流域为中心的区域，大致包括今江西省大部
- **行政区域**：三国时期属孙吴管辖，主要包括豫章郡、庐陵郡、九江郡部分地区
- **历史意义**：重要的农业区和战略要地，控制南北交通要道

### 3. 江南区域
- **地理范围**：长江中下游以南的广阔区域，包括今江苏南部、浙江、江西、湖南、安徽南部等
- **文化意义**：自三国时期开始逐渐发展，至唐宋后成为中国主要的经济文化中心
- **历史发展**：三国时期是江南开发的重要阶段，孙吴政权促进了该地区的农业、手工业和商业发展

### 4. 长江水道
- **地理范围**：重点展示三国时期重要的长江中下游段，特别是荆州至建业之间的关键江段
- **军事意义**：长江是三国时期魏吴对峙的天然分界线，具有重要的军事战略价值
- **经济意义**：长江是重要的水运通道，促进了沿岸地区的经济交流与发展

### 4. 重要城市
- **建业（今南京）**：孙吴都城，政治中心
- **南昌（豫章）**：江西中心，控制赣江水道
- **苏州（吴郡）**：江东重要经济中心
- **绍兴（会稽）**：江东重要文化中心
- **九江**：长江重要渡口和军事要地
- **武昌**：长江中游军事重镇

## 自定义修改

### 1. 添加新的地理区域
在 `three-kingdoms-yangtze-data.js` 文件中，按照以下格式添加新的区域：

```javascript
{
    name: '区域名称',
    coordinates: [[纬度1, 经度1], [纬度2, 经度2], [纬度3, 经度3], ...], // 多边形坐标
    description: '区域描述',
    history: '历史意义',
    dynasty: '朝代',
    ruler: '统治者'
}
```

### 2. 修改地图样式
在 CSS 部分可以修改以下样式：
- 地图容器大小：`#interactive-map` 选择器
- 按钮样式：`button` 选择器
- 图例样式：`.legend` 相关选择器
- 弹窗样式：`.info-popup` 相关选择器

### 3. 调整地图初始视图
在 `three-kingdoms-yangtze-map.html` 文件中，修改以下代码：
```javascript
const map = L.map('interactive-map').setView([30.5, 118.5], 7);
```
其中 `[30.5, 118.5]` 是初始中心坐标，`7` 是初始缩放级别。
