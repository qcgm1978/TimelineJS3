使用说明

静态地图：直接使用图片文件展示地图，适合展示历史地图、自定义地图等。

交互式地图：使用 Google Maps 嵌入，支持缩放、平移、切换地图类型等交互功能。

自定义地图尺寸

在 TimelineJS3 配置中，可以通过以下方式自定义地图显示尺寸：

var options = { width: "100%", height: "600px", maptype: "toner", // 仅适用于 Google Maps lang: "zh-CN" };
